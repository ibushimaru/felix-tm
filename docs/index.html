Felix TM

Translation Memory for Google Sheets — local-first, free, open source.

日本語

What it is

Felix TM is a Chrome extension that brings translation memory (TM) and glossary lookup directly into Google Sheets. As you move between cells, Felix TM matches the source text against your private TM, surfaces fuzzy matches in real time in a side panel, and writes approved translations back into the target column with one click.

Built for translators who work in spreadsheets — game localization, UI string tables, terminology lists, anywhere the unit of translation is a row.

Add to Chrome — coming to the Web Store soon View source on GitHub

Features

Real-time fuzzy matching

Levenshtein-based scoring with bag-distance pre-filtering. Finds 65 %+ matches as you type, faithful port of the original Felix CAT algorithm.

Auto Translate

Apply 100 % matches across a selection in one action. Numbers and registered glossary terms are placed automatically.

Glossary & placement rules

Term-level translations with case preservation. Regex-based placement for product names, version strings, and recurring patterns.

Quality control

Catches mismatched numbers, missing ALL CAPS terms, and untranslated glossary items. Click a flag to register the fix.

TMX / TSV / XLSX I/O

Standard format import and export. Bring your existing TM in or share it across tools.

Local-first & private

Your TM lives in your browser's IndexedDB. No backend, no analytics, no external transmission. Sheets API calls go directly browser → Google.

Privacy at a glance

  • Your TM and glossary stay in your browser. Felix TM has no server.
  • The spreadsheets OAuth scope is used only when you take an explicit action (Auto Translate, Import, etc.).
  • No analytics. No tracking. No third-party scripts.

Full privacy policy.

About Felix CAT

Felix TM is inspired by and shares its matching algorithm with Felix CAT, the Windows-only translation memory tool originally created by Ryan Ginstrom and released as open source under the MIT license. Felix TM is a separate project — a ground-up re-implementation in Python and JavaScript targeting Google Sheets and the modern web — and is not affiliated with Mr. Ginstrom or the original Felix CAT product. We are grateful for the algorithm work that made this project possible.

製品概要

Felix TM は、翻訳メモリ (TM) と用語集検索を Google Sheets 上に直接持ち込む Chrome 拡張機能です。セル間を移動するたびに、原文を非公開の TM と照合し、 あいまい一致候補をサイドパネルにリアルタイムで表示。承認した訳文をワンクリックで 訳文列に書き戻します。

スプレッドシート上で作業する翻訳者向けに設計されています — ゲームローカリゼーション、 UI 文字列、用語リスト、行単位での翻訳作業全般。

Chrome に追加 — Web Store 公開準備中 GitHub でソースを見る

主な機能

リアルタイムあいまい一致

レーベンシュタイン距離 + バッグ距離の事前フィルタによる高速検索。65 % 以上のマッチを瞬時に表示。元 Felix CAT アルゴリズムの忠実な移植。

Auto Translate

選択範囲に 100 % 一致を一括適用。数字と登録済み用語は自動配置されます。

用語集 & 配置ルール

大文字小文字を保ったまま用語単位で訳語を適用。製品名・バージョン文字列・定型パターン用に正規表現ベースの配置ルールも対応。

品質チェック

数字の不一致、訳出漏れの ALL CAPS 用語、未翻訳の用語集項目を検出。フラグをクリックすればその場で用語登録できます。

TMX / TSV / XLSX 入出力

標準フォーマットでのインポート・エクスポート。既存の TM を取り込んで他ツールと共有できます。

ローカル完結・プライバシー

TM はブラウザの IndexedDB 内に保存。サーバーなし、アナリティクスなし、外部送信なし。Sheets API はブラウザから Google へ直接通信。

プライバシーの要点

  • TM・用語集はブラウザ内に留まります。Felix TM にサーバーはありません。
  • spreadsheets OAuth スコープは、Auto Translate やインポート等の明示的操作時にのみ使用されます。
  • アナリティクスなし。トラッキングなし。サードパーティスクリプトなし。

詳細は プライバシーポリシー をご覧ください。

Felix CAT との関係

Felix TM は、Ryan Ginstrom 氏が制作し MIT ライセンスでオープンソース公開した Windows 専用翻訳メモリツール Felix CAT から着想を得ており、 マッチングアルゴリズムを共有しています。Felix TM は独立したプロジェクトであり — Google Sheets と現代の Web 環境を対象に Python と JavaScript で 一から再実装したものです — Ginstrom 氏および元 Felix CAT 製品とは 関係を持ちません。本プロジェクトを可能にしたアルゴリズムへの感謝を申し上げます。
